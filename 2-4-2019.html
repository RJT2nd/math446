Theorem

Assume that g is continuously differentiable, that g(r) = r, and that S = |g'(r)| < 1.
Then the fixed point iteration converges linearly with rate S to the fixed point r for intial guesses sufficiently close to r
ei+1 = S ei

Example

g3(x) = (1+2x^3)/(1+3x^2) root r approx= 0.6823
g'3(x) = (6x^2(1+3x^2) - (1+2x^3)6x) / (1+3x^2)^2
g'3(0.6823) = 0
S3 = 0
FPI converges immediately

g2(x) = (1 - x)^(1/3)
g'2(x) = -1/3 (1 - x)^(-2/3)
|g'2(x)| approx= 0.716 S2 = 0.716
FPI will converge slowly

g1(x) = 1-x^3
g'1(x) = -3x^2
|g'1(0.6823)| = 1.3966
S1 = 1.3966
FPI will not converge

Definition

An iterative method is called locally convergent to r if the method converges to r for initiall guesses sufficiently close to r.
ie, if there exists a neighborhood (r-epsilon, r+epsilon), epsilon > 0, such that convergence to r follows from all initial guesses from that neighborhood.

Example 1.3

Explain why the fixed point iteration converges for g(x) = cos(x)
r = 0.74
g'(x) = -sin(x)
|g'(0.74)| = |-sin(.74)| = 0.67
0.67 < 1 => FPI converges

Stopping Criteria

Option 1: set the number of iteration (kmax)
Option 2: Limit your error
a) Absolute error: |xi+1 - xi| < TOL
b) Relative error: |xi+1 - xi|/|xi+1| < TOL

0.1 Evaluating Polynomials

Objective: Evaluate a polynomial at a certain number with the least number of operations

Example

P(x) = 2x4 + 3x3 - 3x2 + 5x - 1
Evaluate P(x) at x = 1/2.

Method 1

P(1/2) = 2 * 1/2 * 1/2 * 1/2 * 1/2 + 3 * 1/2 * 1/2 * 1/2...
10 multiplications and 4 additions

Method 2

Find the powers and store them
P(1/2) = 2 * (1/2)4 + ...
1/2 * 1/2 = (1/2)2
(1/2)2 * 1/2 = (1/2)3...
7 multiplications and 4 additions

Method 3

Nested Multiplications (inside out)
P(x) = -1 + 5x - 3x2 + 3x3 + 2x4
= -1 + x( 5 - 3x + 3x2 + 2x3)
= -1 + x( 5 - x( 3 + 3x + 2x2))
= -1 + x( 5 - x( 3 + x( 3 + 2x )))
4 multiplications and 4 additions
